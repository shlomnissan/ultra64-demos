
The function and characteristics of the sample program is explained below:

After compiling,  this can be operated in the Partner Command Window by executing 
"1(The file name for debug information)".

ddspgame:	This is a sample program for the N64 Disk Drive.  This will call a Disk
 		function when displaying the screen.  The error result and error message will
 		be displayed using ROM font which is built into the N64 Disk Drive.  By
        	default,  this is a Disk activating program.  In order to activate the Game
 		Pak,  please process the make using "make START_FROM_CASSETTE=".
